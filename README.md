# Student GPA Prediction Project

This project includes a machine learning model used to predict the GPA (Grade Point Average) of students. The model can predict a student's GPA based on factors such as the number of ethnic food types consumed, gender, scone calorie intake, the feeling of being healthy, and the count of exercises.
